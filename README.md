## AXIS

Map that suggests a location is safe/unsafe/warning zone based on risk score for that area/location. Risk score is calculated using votes received from women.

### Tech Stack
- Backend: FastAPI
- Frontend: React
- API: FastAPI
- Maps: GoogleMaps (using API)
- Database: MySQL
- Deployment: Render, Railway
